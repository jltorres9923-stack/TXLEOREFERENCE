# TX LEO — Texas Law Enforcement Reference

A fast, offline-ready field reference app built for Texas law enforcement officers.

## Features

- **Texas Penal Code** — Full offense library with elements, punishment ranges, and enhancement notes
- **Transportation Code** — Traffic violations, commercial vehicle law, and speed tables
- **Controlled Substances (HSC 481)** — Penalty groups with schedules and quantity thresholds
- **DWI / SFST Reference** — HGN, Walk-and-Turn, One-Leg Stand procedures and legal thresholds
- **Landmark Case Law** — Key SCOTUS and Texas appellate decisions on search, seizure, and stops
- **Municipal Codes** — Instant search across 3,841+ Houston action codes
- **Charge Lookup** — Search across all code categories simultaneously
- **Favorites** — Bookmark frequently referenced statutes for fast access
- **Wellness Tools** — Box breathing (4-4-4-4) and 5-4-3-2-1 grounding reset

## Tech Stack

- Swift / SwiftUI
- iOS 18+
- StoreKit 2 (subscription management)
- 100% offline — no network dependency for reference content
